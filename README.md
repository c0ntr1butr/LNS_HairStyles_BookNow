# LNS HairStyles: Book Your Appointment

A simple QR-based salon appointment booking website for LNS HairStyles.

Customers can scan a QR code, open the website, fill appointment details, and send the booking request to the salon owner through WhatsApp.

## Features

- Mobile-friendly booking form
- Service selection
- Date and time selection
- Customer contact details
- WhatsApp booking confirmation
- Google Sheets backend support
- Email notification support through Google Apps Script
